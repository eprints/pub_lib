# Research Publications Flavour (pub_lib)

The Research Publications ('pub_lib') flavour is designed to extend EPrints 3.5 to make it into a repository suitable for storing research publications.\
It adds extra fields to the eprint data object, bespoke import and export plugins and other functionality and configuration tailored for research publications.\

The Research Publications flavour comes packaged by default with EPrints 3.5, as it is the typical use case for an EPrints repository.\
Please see https://github.com/eprints/eprints3.5 for more information.
